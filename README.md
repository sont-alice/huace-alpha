# A股波段预测推荐软件

本项目是一个本地 Web 软件原型，用于 A 股 2-6 周波段推荐研究。它先做回测验证，再展示推荐；没有通过门槛时允许输出“今日无推荐”。

当前版本默认使用内置样例数据，方便在无账号、无外网数据依赖时跑通完整流程。安装 `akshare` 或 `tushare` 后，可继续扩展真实数据适配器。

## 启动

```powershell
streamlit run app.py
```

## 测试

```powershell
pytest
```

## 重要说明

软件仅用于量化研究和辅助决策，不构成投资建议，不承诺收益或固定胜率。样例数据模式下的推荐不是现实市场推荐。
