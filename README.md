# A股波段预测推荐软件

本项目是一个本地 Web 软件原型，用于 A 股 2-6 周波段推荐研究。它先做回测验证，再展示推荐；没有通过门槛时输出“仅观察”或“今日无推荐”。

当前版本默认拉取真实数据：

- AKShare：默认数据源，拉取核心股票池的前复权日线、上市信息、行业字段和财务指标。
- Tushare Pro：填写 token 且勾选优先使用后启用，拉取日线、日行情基础指标和财务指标。
- 样例数据：仅在勾选“演示模式”或真实数据源不可用时使用。

真实数据会缓存到 `data/cache/`，同一天相同参数会优先读取缓存。

## 功能

- 今日推荐：展示通过模型排序和风险过滤后的波段候选。
- 个股评估：输入股票代码后，输出模型分位、技术面、基本面、风险标签、操作结论和近一年走势。
- 板块筛选：支持上证主板、深证主板、创业板、科创板；输入评估代码时会强制加入该股票，不受默认股票数量限制。
- 数据兜底：公开接口不可用时会回退样例模式；输入代码会生成占位样例，避免界面中断，但占位结果不能用于现实判断。
- 回测证据：展示样本外资金曲线、胜率、回撤和收益回撤比。
- 数据源状态：展示 AKShare/Tushare 可用性、数据行数和训练/测试切分。

## 安装

```powershell
python -m pip install -e .
```

## 启动

```powershell
streamlit run app.py
```

## 测试

```powershell
pytest
```

## 重要说明

软件仅用于量化研究和辅助决策，不构成投资建议，不承诺收益或固定胜率。公开数据接口可能受网络、频次、权限和字段变动影响；界面会显示当前数据源状态。
