# 胜率提升与商业化界面设计规格

日期：2026-06-13

## 目标

在现有 A 股 2-6 周波段推荐软件基础上，提高推荐质量和专业使用体验。胜率提升必须通过可解释的策略增强、样本外回测和风险过滤实现，不承诺固定收益或固定胜率。

## 策略增强

- 从单一回归模型升级为多模型集成排序。
- 增加市场状态和市场宽度特征，避免在弱市中机械推荐。
- 增加趋势评分、风险评分、基本面评分、行业评分和综合评级。
- 增加买入门槛：模型分位、趋势状态、流动性、波动、行业强度、市场状态同时满足时才给出“买入观察”。
- 对历史失败特征进行反向过滤：高波动、弱行业、低流动性、低分位的股票降级。
- 回测仍使用同一套推荐逻辑，避免线上推荐和历史验证不一致。

## 产品体验

- 改为成熟金融终端风格：深色背景、冷灰面板、蓝色主操作、绿色收益、红色风险、金色评级。
- 首屏为“今日工作台”：市场状态、模型质量、推荐数量、风险摘要、推荐表。
- 个股评估页展示结论、评分拆解、趋势图、风险标签和操作解释。
- 侧栏集中放置策略、个股、数据源和板块设置。

## 验收

- 测试通过。
- 真实 AKShare 小样本可跑通。
- 页面可通过 `http://localhost:8501` 打开。
- 推荐表展示综合评级和评分拆解。
- 个股评估展示商业化评分卡和结论。
